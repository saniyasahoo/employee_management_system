# Employee Management System (EMS)

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- Sequelize ORM
- Postman
- Express File Upload

## Features
- Department CRUD
- Designation CRUD
- Employee CRUD
- Experience Management
- Image/File Upload
- REST API based architecture

## How to Run
1. Clone the repo
2. Install dependencies: npm install
3. Configure PostgreSQL in .env
4. Run: npm start
5. Test APIs using Postman
